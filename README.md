# IonicApp-BlahBlahCar

IonicApp-BlahBlahCar is a simple cordova application using Ionic and Firebase.


### Launch app on web browser

    ionic serve

## Features

    - Authentification 
        - Register Users.
        - LogIn Users.
        - Reset Password.
        - Logout.
        - Form Validation Ready.
